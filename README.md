# Loan Approval Prediction Web App

A Machine Learning web application that predicts whether a loan will be approved based on applicant details.

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Features
- Real-time prediction
- Model comparison
- Confidence score
- Decision explanation

## Run Locally

1. Install libraries
pip install -r requirements.txt

2. Run the app
streamlit run app.py
